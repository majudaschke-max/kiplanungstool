# KI-Einsatz sinnvoll?

Diese Mini-App unterstützt Lehrkräfte bei der Planung und Reflexion von KI-Einsatz im Unterricht.

Sie hilft dabei zu prüfen, ob KI den Lernprozess sinnvoll unterstützt, ob die Schülerinnen und Schüler selbst aktiv bleiben und ob die eigene Arbeit sichtbar wird. Die App gibt konkrete Planungshinweise, Risiken, Verbesserungsimpulse sowie Prompts für KI-Assistenten im Unterricht aus.

## Nutzung

Die App läuft direkt im Browser:

1. Repository öffnen
2. `index.html` starten oder die GitHub-Pages-Version nutzen
3. Keine Installation nötig

Bitte keine personenbezogenen Schülerdaten eingeben.
